​L'implementation guide contient un package téléchargeable ici permettant de valider les instances par rapport aux profils qu'il contient.

​Pour cela, il suffit de télécharger le package.tgz et l'importer dans un serveur, par exemple sur hapi en suivant ce script python open source.

Vous pourrez ensuite utiliser l'opération $validate pour valider les instances de ressource contre un profil issu de cette spécification.
