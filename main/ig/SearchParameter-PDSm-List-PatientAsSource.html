SearchParameter: PDSm-List-PatientAsSource

Official URL: https://interop.esante.gouv.fr/ig/fhir/pdsm/SearchParameter/PDSm-List-PatientAsSource Version: 3.1.1
Active as of 2021-07-30 Computable Name: PDSmListPatientAsSource

Paramètre de recherche chaîné créé pour le volet PDSm de manière à pouvoir utiliser comme critère de recherche l'élément source:Patient.given et source:Patient.family

PDSmListPatientAsSource

Paramètre source:reference

Paramètre de recherche chaîné créé pour le volet PDSm de manière à pouvoir utiliser comme critère de recherche l'élément source:Patient.given et source:Patient.family

RessourceList
ExpressionList.source
Mode de traitementNormal
Ressources ciblesPatient
Multiples
  • multipleAnd : Le paramètre peut se répéter pour spécifier plusieurs valeurs qui doivent toutes être vraies
  • multipleOr : Le paramètre peut avoir plusieurs valeurs (séparées par des virgules) dont au moins une doit être vraie
ModificateursAutorisé : exact, contains
ChaînesAutorisé : given, family
